PokerVision

A computer vision system for recognizing poker cards from screen captures using machine learning, specifically designed for PokerStars Texas Hold'em 6-player tables.

Overview

PokerVision uses computer vision and machine learning techniques to recognize poker cards directly from screen captures. The system can identify card ranks and suits in real-time, track table cards, analyze hand strength, and estimate win probability. It is optimized for PokerStars Texas Hold'em 6-player tables.

Features

  • Screen capture functionality to grab card regions
  • Card detection using computer vision
  • Card rank and suit recognition using trained neural networks
  • Empty position detection to track game progress
  • Real-time display interface built with Pygame
  • Support for Texas Hold'em cards (2-9, 10, J, Q, K, A)
  • Hand evaluation and win probability calculation
  • Game state tracking (Pre-flop, Flop, Turn, River)
  • Optimized for PokerStars UI and 6-player table layout

Prerequisites

  • Python 3.7+
  • OpenCV
  • TensorFlow 2.x
  • Pygame
  • NumPy
  • scikit-learn
  • Matplotlib
  • Seaborn
  • PokerStars desktop client

Installation

  1. Clone the repository: 
    git clone https://github.com/Adameelmadani/PokerVision.git
cd PokerVision
  2. Install virtual environment: 
    pip install virtualenv
  3. Create a virtual environment: 
    virtualenv -p python env
  4. Activate the virtual environment: 
    env\Scripts\activate.bat
  5. Install dependencies: 
    pip install -r requirements.txt

Dataset Structure

  • /data/cards_numbers/ - Card ranks like r_2.png, b_10.png
  • /data/cards_suits/ - Card suits like Clubs_1.png, Diamonds_2.png
  • /data/empty_positions/ - Empty card position images like rank_pos1.png

Training the Models

python train_model.py
python test_models.py

Usage

  1. Launch PokerStars and join a 6-player table.
  2. Run: 
    python main.py
  3. The app will detect cards and track the game in real-time.
  4. Edit main.py for screen region adjustments.

Project Structure

  • main.py - Main app with Pygame interface
  • screenshot.py - Screen capture tools
  • card_detector.py - Card detection logic
  • card_recognizer.py - Model-based recognition
  • train_model.py - Model training scripts
  • test_models.py - Evaluation scripts
  • poker_evaluator.py - Hand evaluation logic
  • /data/ - Training/test images
  • /models/ - Trained models
  • /evaluation/ - Metrics & visualizations

Configuration

  • Edit coordinates in main.py for your resolution
  • Default optimized for 6-player tables
  • Use screenshots from your PokerStars theme
  • Train models with your images
  • Fine-tune detection as needed

Troubleshooting

Models not detecting? Check screen region, use more data, test with test_models.py

Slow performance? Reduce capture resolution or detection frequency
