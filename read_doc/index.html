PokerVision Documentation

PokerVision is a tool for recognition of all entities on the poker table (currently only for PokerStars), with analytics to help you make strategic decisions about your moves.

NOTE: This tool only works with the specific theme included and for 6-player tables. It has been tested on Ubuntu with Python 3.8.

Installation

Follow these steps to install and run PokerVision:

git clone https://github.com/yourusername/pokervision.git
cd pokervision
pip install -r requirements.txt

Usage

Run the main script to start table recognition and analytics:

python main.py

Components

Table Recognition

Identifies cards, players, and chips using OpenCV and template matching on PokerStars tables.

Equity Calculation

Calculates your win probability based on your cards and the table state using pypokerengine.

API Reference

Equity Module

Provides functions to calculate the equity of hands. Main functions include:

  • calculate_equity(hand, board)
  • simulate_outcomes()

Recognition Module

Handles image capture and element detection from the PokerStars interface. Example functions:

  • detect_cards()
  • detect_players()

Utilities

Helper functions for data manipulation and formatting:

  • format_chips(amount)
  • get_screenshot()

Configuration

Edit config.yaml to set recognition thresholds, table positions, or theme preferences.
